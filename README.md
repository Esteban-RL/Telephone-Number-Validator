# Telephone-Number-Validator
A US Telephone number validator with regex
This app can validate the following formats:
1 555-555-5555;
1 (555) 555-5555;
1(555)555-5555;
1 555 555 5555;
5555555555;
555-555-5555;
(555)555-5555;
